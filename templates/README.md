## How to Use

1. **Download** the template you want
2. **Open** in any text editor
3. **Customize** the color scheme (see top of `<style>` section)
4. **Replace** placeholder text with your content
5. **Add** your images (replace emoji placeholders)
6. **Upload** to any web host

## Customization Guide

### Change Colors
Look for the `:root` section at the top of the CSS:
```css
:root {
    --primary: #your-color;
    --secondary: #your-color;
    /* etc. */
}
```

### Replace Images
- Hero backgrounds: Change the `url()` in `.hero` section
- Gallery/portfolio images: Replace emoji placeholders with `<img>` tags
- Use your own photos or free stock photos from Unsplash/Pexels

### Edit Content
- All text is clearly labeled in the HTML
- Update business name, contact info, prices, etc.
- Remove sections you don't need (just delete the `<section>` block)
